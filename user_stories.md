## User Stories

- As a user, I want to register so I can submit posts.
- As a user, I want to log in so I can vote and comment.
- As a user, I want to log out securely.

#### Posts (Stories)

- As a user, I want to see a list of latest stories.
- As a user, I want to see top-ranked stories.
- As a user, I want to submit text-only posts.

#### Voting
- As a user, I want to upvote a post.
- As a user, I want to upvote a comment.
- As a user, I should be able to vote only once per item.

#### Comments

- As a user, I want to comment on a post.
- As a user, I want to reply to another comment.


#### User Profile

- As a user, I want to view my profile.
- As a user, I want to see my submitted posts.

#### Jobs

- As a user, I want to view Jobs.
